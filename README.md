# e-Manifest Data Upload Tool
This application uploads data from a Microsoft Excel workbook to EPA's e-Manifest system via REST API. It allows creation of multiple electronic manifests in the same batch using data exported from another system or database.

## Requirements
NodeJS [Download](https://nodejs.org/en/download)

## Installation
```shell
npm install
```

## Running the Application
```shell
npm start
```

If making changes to the frontend (HTML or JS), make sure to run the below so the CSS file is updated.
```shell
npm run watch:css
```

## Building
```shell
npm run make
```
